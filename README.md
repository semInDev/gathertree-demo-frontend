# 🎄 GatherTree Frontend (Example)

이 프로젝트는 크리스마스 트리 꾸미기 예시용 프론트엔드입니다.
(Figma 대신 동작 예시 공유용)

## 🚀 실행 방법

### 1. Node.js 버전

- Node.js 18 이상 권장


### 2. 의존성 설치

```bash
npm install
```

### 3. 로컬 실행

```bash
npm run dev
```

### 4. 접속

브라우저에서 아래 주소로 접속
[http://localhost:5173](http://localhost:5173)
